# Redis

## 简介

Redis：Remote Dictionary Server

Redis是一个开源（BSD协议）的高性能 Key-Value 型内存数据库。

Redis是一种NoSQL数据库（非关系型数据库）。

## 官网

官网地址：redis.io



## 版本

目前（2020-10）Redis 最新稳定版本为 6.0.x 





## 为什么使用Redis作为缓存

- 高性能  （单机10W QPS）
- 高可用  （主从、哨兵、集群）
- 支持多种数据结构
- 单线程工作模式，并发安全；采用IO多路复用机制
- 支持数据持久化、事务、订阅发布等多种功能
